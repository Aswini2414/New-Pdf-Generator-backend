# Mern stack Project: NEW PDF GENERATOR BACKEND

## Run Locally

### Setup .env file

```
DATABASE_URL = .....
```

#### Clone the Project
```
git clone https://github.com/Aswini2414/New-Pdf-Generator-backend.git
```

#### Go to the Project directory
#### Install dependencies
```
npm install
```
#### Start the server
```
npm start
```
